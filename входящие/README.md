# Входящие материалы

Сюда кладёшь сырые файлы: расшифровки голосовых, массивы данных, заметки.

## Как работает

- **Новый файл** → просто сохрани в эту папку
- **После обработки** → Claude перемещает файл в `входящие/обработано/`

## Как запустить обработку

Скажи Claude: "обработай входящие" — он найдёт все файлы в корне этой папки, которых нет в `обработано/`, и интегрирует знания в wiki.

## Формат файлов

Называй файлы понятно, дата необязательна но полезна:
- `2026-05-10 голосовая про онбординг.md`
- `процессы на посте.md`
- `история команды.md`
