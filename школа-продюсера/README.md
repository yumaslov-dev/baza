# Школа продюсера — прототип

_Обновлено: 2026-06-20_

Прототип отдельного продуктового маршрута на основе материалов BAZA.

Логика: провести человека от понимания продюсерской роли к самостоятельному ведению клиента, проекта, команды и сложных разговоров.

## Файлы

- `mvp.html` — вторая версия прототипа: продуктовый вход, диагностика, ситуации, авторский метод и блок рыночной проверки.
- `index.html` — первая плотная HTML-страница курса с 18 подробными карточками.
- [[курс]] — структура курса и список карточек.

## Как смотреть

Начинать лучше с `mvp.html`. Это прототип для разговора с рынком: его можно показать продюсерам, редакторам, маркетологам и потенциальным участникам, чтобы проверить, где они узнают себя и какой формат им нужен.

`index.html` стоит использовать как контентную базу: там больше подробных карточек, но UX пока плотнее.

## Принцип сборки

Карточки собраны по подробной логике раздела [[продюсер-вход-в-задачу]]:

1. Ситуация.
2. Почему важно.
3. Где обычно ломается.
4. Как смотреть точнее.
5. Что делать.
6. Примеры, фразы или шаблоны.
7. Артефакт.
8. Критерий хорошего результата.
9. Главное правило.
