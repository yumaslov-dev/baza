# Входящие статьи

⛔ **Тексты в этой папке не редактируются и не корректируются.**

Сюда кладутся статьи в том виде, в котором они были получены — черновики, копии, расшифровки, чужие материалы.

## Правило

Claude **не изменяет** содержимое файлов в этой папке.  
Допустимые действия: **только читать** и **анализировать** для последующей работы.

## Как использовать

- Сохрани статью как `.md`-файл в эту папку
- Скажи Claude что нужно сделать: разобрать структуру, написать конспект, создать черновик на основе, и т.д.
- Результат работы Claude помещает в `Статьи/` (рядом с этой папкой), исходник остаётся нетронутым

## Формат имён файлов

Называй понятно, дата по желанию:
- `2026-05-22 статья про продюсирование.md`
- `материал из телеграма.md`
- `черновик-интервью.md`
