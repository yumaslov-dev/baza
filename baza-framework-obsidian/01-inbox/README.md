# Входящие

Сюда кладутся сырые заметки, расшифровки, проектные ситуации и мысли для параллельной базы.

## Как обрабатывать

1. Прочитать материал.
2. Найти рабочие ситуации, процессы, принципы, шаблоны и кейсы.
3. Создать отдельные файлы в `02-cards`, `03-processes`, `04-templates` или `05-principles`.
4. Проставить frontmatter, связи, статус и видимость.
5. Обновить `06-site/index.md`, если появился новый готовый материал.

## Правило

Один файл во входящих может дать несколько карточек. Одна карточка решает одну рабочую ситуацию.
