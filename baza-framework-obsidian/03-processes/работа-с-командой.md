# Работа с командой

## Когда использовать

Когда задача зависит от нескольких людей и продюсеру нужно сохранить ясность, ответственность и рабочий темп.

## Цикл работы

1. Поставить задачу через контекст, результат и критерии.
2. Договориться о первой точке проверки.
3. Проверять работу по фактам, а не по тревоге.
4. Помогать через диагностику стопора.
5. Давать обратную связь по работе.
6. Разбирать ошибки как цепочку событий.
7. Фиксировать новые правила.

## Артефакт

Письменные задачи, точки проверки, правила после ошибок и договорённости по ответственности.
