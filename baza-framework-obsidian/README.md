# База знаний продюсера

Стартовый пакет для базы знаний MASS AGENCY в связке Obsidian + Claude.

## Как устроен пакет

- `00-system` — правила системы, стиль, таксономия, инструкции для Claude.
- `01-inbox` — место для сырых заметок, расшифровок, мыслей и материалов.
- `02-cards` — готовые карточки знаний.
- `04-templates` — шаблоны для рабочих артефактов.
- `06-site` — структура главной страницы и будущей витрины базы.

## Главный принцип

База должна помогать человеку понять рабочую ситуацию, принять решение и сделать следующий шаг.

Каждый материал отвечает на три вопроса:

1. Что происходит?
2. Почему это важно?
3. Что делать дальше?

## Базовый процесс

1. Всё новое складывается в `01-inbox`.
2. Claude разбирает сырьё по правилам из `00-system/CLAUDE.md`.
3. Из сырья создаются карточки, кейсы, процессы или шаблоны.
4. Готовые материалы получают статус `ready`.
5. Материалы со статусом `ready` или `published` попадают на сайт.

## Рекомендуемый порядок запуска

1. Прочитать `00-system/DESIGN-SYSTEM.md`.
2. Проверить разделы и типы в `00-system/TAXONOMY.md`.
3. Использовать `00-system/CARD-TEMPLATE.md` для первых карточек.
4. Отдать Claude правила из `00-system/CLAUDE.md`.
5. Собрать главную по `00-system/HOMEPAGE-FRAMEWORK.md`.
