Как выбирать подрядчика
8 вопросов перед выбором
Выбор
›

Если подрядчик новый — нельзя сразу отдавать критически важную часть проекта без проверки.

8 вопросов перед выбором

  1. Какая задача и уровень ответственности?
  2. Какой бюджет и срок?
  3. Нужна самостоятельность или плотный контроль?
  4. Был ли опыт с этим человеком?
  5. Кто из команды может поручиться?
  6. Есть ли риск по срокам, качеству или коммуникации?
  7. Что будет, если сорвётся?
  8. Есть ли запасной вариант?

Алгоритм проверки нового подрядчика

  1. Дать небольшую задачу
  2. Составить понятный бриф
  3. Установить ограниченный срок
  4. Провести промежуточную проверку
  5. Оценить результат
  6. Зафиксировать вывод в базе
